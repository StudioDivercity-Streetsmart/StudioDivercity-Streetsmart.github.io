## Deployment
This is deployed on GitHub Pages, so simply commit any changes you'd like to `main` and they will automatically publish to the internet. 

## Preview deployments
To get a preview deployment, simply:

1. commit and push a branch (you need to have edit access to the repo)
2. create a PR (there is a nice notification that will appear when you log into GitHub)
3. the bot will comment with a link to your preview deployment after a minute or two
